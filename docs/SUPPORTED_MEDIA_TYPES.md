## Input Media Types

### Video Files

- `.mp4`, `.mkv`, `.avi`, `.mov`

### Audio Files

- `.mp3`, `.wav`, `.m4a`, `.aac`

### Text Files

- `.txt`

### Image Files

- `.png`, `.jpg`, `.jpeg`, `.gif`, `.bmp`, `.tiff`, `.tif`, `.webp`

## Output Formats

- **Transcript**: `.txt` - Extracted text from audio/video/images
- **Study Material**: `.md` - Structured study materials with summaries, glossaries, and questions
- **PDF**: `.pdf` - Formatted PDF documents with AI-generated content (output only)

**Note**: PDF files are generated as output and are not supported as input media.
